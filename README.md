## Getting Started

To use it locally First clone, install packages and run the development server:

```bash
git clone https://github.com/ZaminMirzad/movie-website-clone.git
yarn
yarn dev
# or
pnpm dev
```

And it will run on [https://localhost:3000](https://localhost:3000)
